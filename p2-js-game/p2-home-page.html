HOW TO PLAY

1. Random questions about JavaScript facts will be provided.
2. Answer using True or False buttons.
3. After you answer, it will show you the correct answer.
4. Then, next question button will pop-out. Click it if you're ready to go to the next question.
5. There will be 30 questions in total, the game will automatically end once you've finish answering all questions.
6. You will see how many correct answer you have in 'SCORE'.
7. You don't need a perfect score, but since this will test how deep you know JavaScript, if you can get over 21 correct answers, it means you do know JavaScript well.
8. Feel free to click the 'RESTART' button to restart the quiz.
9. Feel free to click the "QUIT GAME" button if you think you're not yet ready for the quiz.
